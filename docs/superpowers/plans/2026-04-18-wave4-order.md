# Order Module — Wave 4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` or `superpowers:executing-plans`. Steps use `- [ ]` syntax.

**Goal:** Implement the `order` Gradle module — a Core Domain (Hexagonal/Ports & Adapters) providing tenant-scoped order management with a state machine (`PENDING → CONFIRMED → SHIPPED → DELIVERED → CANCELLED`), aggregate root protecting state transitions, domain events, and full CRUD endpoints.

**Architecture:** Core Domain → strict Hexagonal layout (same as `product`). `Order` aggregate root owns `OrderItem` value objects. State transitions are enforced by domain methods: `confirm()`, `ship()`, `deliver()`, `cancel()`. Domain events: `OrderCreated`, `OrderCancelled` (published after commit). `order_items` is a child table — fetched eagerly alongside `orders`. ArchUnit test enforces domain isolation. No payment integration in this wave.

**Tech Stack:** Java 25, Spring Boot 4, jOOQ 3.20, Spring Modulith events, ArchUnit.

**Prerequisites:**
- Wave 1 complete (tenant + RLS).
- Wave 2a complete (User CRUD, users.status, UserRepository updated).
- Wave 3a complete (Product module — `product.api.ProductId` referenced in order items).
- Wave 3b (Member) does NOT need to complete before Wave 4.
- `./gradlew check` all green before starting.

    id              UUID        PRIMARY KEY,
    tenant_id       UUID        NOT NULL REFERENCES tenants(id),
    user_id         UUID        NOT NULL REFERENCES users(id),
    status          VARCHAR(20) NOT NULL DEFAULT 'PENDING',

CREATE TABLE order_items (
    id              UUID        PRIMARY KEY,
    order_id        UUID        NOT NULL REFERENCES orders(id) ON DELETE CASCADE,
    product_id      UUID        NOT NULL,  -- logical FK to products(id); no hard FK (cross-module)
    product_code    VARCHAR(100) NOT NULL,  -- snapshot at order time
    product_name    VARCHAR(200) NOT NULL,  -- snapshot at order time
    unit_price      NUMERIC(19,4) NOT NULL CHECK (unit_price >= 0),
    currency        VARCHAR(3)  NOT NULL DEFAULT 'CNY',
    quantity        INTEGER     NOT NULL CHECK (quantity >= 1),
    subtotal        NUMERIC(19,4) NOT NULL CHECK (subtotal >= 0)
);

COMMENT ON TABLE orders      IS 'Order aggregate root. Status machine: PENDING→CONFIRMED→SHIPPED→DELIVERED|CANCELLED.';
COMMENT ON COLUMN order_items.product_code IS 'Snapshot of product code at order creation — survives product rename.';

`Order.java` (aggregate root — no framework imports):
```java
public final class Order {
    private final OrderId id;
    private final UUID tenantId;
    private final UUID userId;
    private final List<OrderItem> items;    // unmodifiable
    private final OrderStatus status;
    private final Money totalAmount;
    private final String note;

    private Order(OrderId id, UUID tenantId, UUID userId, List<OrderItem> items,
                  OrderStatus status, Money totalAmount, String note) {
        this.id = Objects.requireNonNull(id);
        this.tenantId = Objects.requireNonNull(tenantId);
        this.userId = Objects.requireNonNull(userId);
        this.items = List.copyOf(Objects.requireNonNull(items));
        this.status = Objects.requireNonNull(status);
        this.totalAmount = Objects.requireNonNull(totalAmount);
        this.note = note;
        if (items.isEmpty()) throw new IllegalArgumentException("order must have at least one item");
    }

    public static Order create(UUID tenantId, UUID userId, List<OrderItem> items, String note) {
        // Derive currency from first item (all items must share same currency — validate upfront)
    Currency currency = items.get(0).subtotal().currency();
    Money total = items.stream()
            .map(OrderItem::subtotal)
            .reduce(Money.zero(currency), Money::add);
        return new Order(OrderId.newId(), tenantId, userId, items, OrderStatus.PENDING, total, note);
    }

    public static Order reconstitute(OrderId id, UUID tenantId, UUID userId, List<OrderItem> items,
                                      OrderStatus status, Money totalAmount, String note) {
        return new Order(id, tenantId, userId, items, status, totalAmount, note);
    }

    private Order transition(OrderStatus target, OrderStatus... allowed) {
        for (OrderStatus a : allowed) { if (status == a) return withStatus(target); }
        throw new InvalidOrderTransitionException(status, target);
    }

    private Order withStatus(OrderStatus newStatus) {
        return new Order(id, tenantId, userId, items, newStatus, totalAmount, note);
    }

    public Order confirm()  { return transition(OrderStatus.CONFIRMED, OrderStatus.PENDING); }
    public Order ship()     { return transition(OrderStatus.SHIPPED,   OrderStatus.CONFIRMED); }
    public Order deliver()  { return transition(OrderStatus.DELIVERED, OrderStatus.SHIPPED); }
    public Order cancel()   { return transition(OrderStatus.CANCELLED, OrderStatus.PENDING, OrderStatus.CONFIRMED); }

    // Accessors
    public OrderId id()            { return id; }
    public UUID tenantId()         { return tenantId; }
    public UUID userId()           { return userId; }
    public List<OrderItem> items() { return items; }
    public OrderStatus status()    { return status; }
    public Money totalAmount()     { return totalAmount; }
    public String note()           { return note; }
}
```

Key persistence note: `Order` + `OrderItem` involve two tables. `save()` must:
1. UPSERT into `orders` (on conflict by id, update status/updated_at).
2. DELETE existing `order_items` for this order_id, then INSERT all current items.
   (Simple approach — avoid complex diff logic; order items don't change after creation in v1.)

```java
@Override
@Transactional
public void save(Order order) {
    // 1. UPSERT order header
    OffsetDateTime now = OffsetDateTime.now(ZoneOffset.UTC);
    dsl.insertInto(Tables.ORDERS)
        .set(/* all columns */)
        .onConflict(Tables.ORDERS.ID)
        .doUpdate()
        .set(Tables.ORDERS.STATUS, order.status().name())
        .set(Tables.ORDERS.UPDATED_AT, now)
        .execute();
    // 2. Replace items (items are immutable after creation in v1 so this is safe)
    dsl.deleteFrom(Tables.ORDER_ITEMS)
        .where(Tables.ORDER_ITEMS.ORDER_ID.eq(order.id().value()))
        .execute();
    for (OrderItem item : order.items()) {
        dsl.insertInto(Tables.ORDER_ITEMS).set(/* item columns */).execute();
    }
}
```

`findById()` must JOIN `order_items`:
```java
// Fetch order header
OrdersRecord header = dsl.selectFrom(Tables.ORDERS)
    .where(Tables.ORDERS.ID.eq(id.value()))
    .fetchOne();
if (header == null) return Optional.empty();
// Fetch items
List<OrderItemsRecord> itemRecords = dsl.selectFrom(Tables.ORDER_ITEMS)
    .where(Tables.ORDER_ITEMS.ORDER_ID.eq(id.value()))
    .fetch();
return Optional.of(mapper.toAggregate(header, itemRecords));
```

public Order execute(Command cmd) {

    List<OrderItem> items = cmd.items().stream().map(i -> {
